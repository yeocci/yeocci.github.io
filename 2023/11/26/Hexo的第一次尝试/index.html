Hexo的第一次尝试

2023-11-26

引言

很长一段时间，我都没有使用文字记录自己所做过的事情和所学过的知识。随着时间的推移，很多记忆在脑海中慢慢消失，突然某一天想着不能再这样下去了，一定要花点时间记录自己所知的一切，哪怕是短短几行字，相信未来重新读起这些记录，自己也会欣慰。然后就有了今天亿点时间搭建 Hexo 的记录。

搭建过程

前置条件

  • Hexo 文档: 非常有用

  • 一台能够联网并能用于开发设备

  • 一颗还能够使用的大脑🧠

过程

详细查看文档之后，选择将 Hexo 部署到自己的 Github 仓库，然后一顿操作，弄好了🤣。本次搭建使用的主题是 Claudia，整体效果不错，就是暗色主题下，有些文字对比度不够强，后面在看看能不能自己改动颜色。
